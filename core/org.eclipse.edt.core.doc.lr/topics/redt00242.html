DateTimeLib.dateFromInt function

DateTimeLib.dateFromInt returns a DATE value corresponding to an integer value that gives the number of days before or after 31 December 1889.

Package

eglx.lang

Syntax
	static function dateFromInt(numberOfDays EInt in) returns (EDate);
numberOfDays
Number of days before or after 31 December 1889. The number can be positive or negative.
Example use

After the following code runs, the value of result is 1 January 1890.

result Date = DateTimeLib.dateFromInt(1);
